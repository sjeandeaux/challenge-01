# presentator

> it is a refential for polygons.

## Requirements

### tools

- make
- go
- docker
- docker-compose

## How do we play with it

You should install some go tools:

```bash
make download
make help
```

### Run the application

Generate the binary: `go build .`
Run the binary: 

```sh
referentialator -h
Usage of referentialator:
  -http-port int
        The http port (default 9090)
```
